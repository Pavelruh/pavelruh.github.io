Wall Organizer

Project Description

I wanted to use up some filament and tidy up my desk a bit, to find a place for notebooks/sketchbooks while keeping them easily accessible. Created a pocket for notebooks, pencils and erasers. In the end, it fit almost everything from my pencil case plus a caliper.

Implementation

Modeled in Fusion, of course. Initially planned to only fit notebooks, a pencil and an eraser, but I made the pocket quite spacious, and it ended up holding many more items.

3D model
Download 3D model

Also tried to make the plain walls more interesting - added hexagon patterns to the model and used fuzzy skin in the slicer. Had never tried this before. Looks cool and makes the 3D printing layers less visible.

Fuzzy skin texture

This was also a printer test to see if it could remove a large model from the bed. But it started peeling off before the print was finished, so the test wasn't very conclusive.

AutoEjector для OctoPrint

Описание проекта

Проект представляет собой автоматическую систему сброса 3D-модели после завершения печати на принтере Prusa MK3S. Я адаптировал и существенно переработал существующий проект, реализовав его в виде полноценного плагина для OctoPrint на Python. Также были внесены изменения в прошивку Arduino, управляющую механизмом сброса.

Реализация

Помимо программирования, я использовал навыки 3D-моделирования для разработки и печати механической части устройства, а также применил базовые знания электроники при сборке схемы управления. Проект значительно ускоряет процесс серийной печати, устраняя необходимость ручного вмешательства между задачами.

Использованные технологии

  • Python (разработка плагина для OctoPrint)
  • Arduino (C++) (прошивка для управления устройством)
  • OctoPrint API
  • 3D-моделирование (Blender/Fusion 360)
  • 3D-печать
  • Электроника и пайка
  • Проектирование схем
